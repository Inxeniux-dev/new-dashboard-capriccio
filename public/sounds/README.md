# Archivos de sonido para notificaciones

Este directorio debe contener los siguientes archivos de audio para las notificaciones:

- `notification.mp3` - Sonido general de notificación
- `urgent.mp3` - Sonido para notificaciones urgentes (asistencia humana requerida)
- `assigned.mp3` - Sonido cuando se asigna un agente
- `new-chat.mp3` - Sonido para nueva conversación

## Formatos recomendados:
- MP3 con bitrate de 128kbps
- Duración máxima de 2 segundos
- Volumen normalizado

## Recursos gratuitos para sonidos:
- [Freesound.org](https://freesound.org)
- [Zapsplat.com](https://www.zapsplat.com)
- [Notification Sounds](https://notificationsounds.com)

Asegúrate de que los archivos tengan los nombres exactos mencionados arriba.
